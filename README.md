# 📦 Inventory Management CRUD Application

A simple Inventory Management CRUD application built using **Node.js** and **Express.js** for learning and practice purposes. This project uses an **array as a temporary data store** to understand the fundamentals of CRUD (Create, Read, Update, Delete) operations before working with databases.

## ✨ Features

- Add New Inventory Items
- View All Items
- View Single Item Details
- Update Item Information
- Delete Items
- Server-side Rendering
- Simple and Beginner-Friendly Project
- No Database Required

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- EJS

### Backend
- Node.js
- Express.js

## 🎯 Purpose of This Project

This project was created to learn and practice:

- Node.js Basics
- Express.js Fundamentals
- Routing in Express
- CRUD Operations
- Handling Form Data
- Dynamic Rendering with EJS
- Managing Data Using Arrays

## 📌 Note

This project uses an in-memory array to store inventory data. The data is not permanently saved and will be reset whenever the server restarts. The main goal of this project is to understand how CRUD operations work before integrating a database such as MongoDB.
